# 🇯🇵 日本投资现金流计算工具（Cashflow Japan）

## 这个项目是做什么的？

这是一个**专为在日本生活的个人 / 自由职业者设计的现金流计算工具**，帮助你清楚地算出：

* 每个月最多可以投资多少钱
* 投资后，未来能不能稳定提取现金流
* 在市场下跌时，如何避免“越提越穷”

本项目不讲理论、不卖焦虑，只做**能直接用的表格和规则**。

---

## 适合谁使用？

* 在日本工作的上班族 / 自由职业者
* 使用投信、ETF进行长期投资的人
* 希望未来每月有 **3～5 万日元稳定现金流** 的人
* 已经开始投资，但对“能不能长期提取”心里没底的人

👉 **不需要金融背景，也不需要懂编程**。

---

## 你能解决哪些现实问题？

通过本工具，你可以：

* ✅ 算清楚「每月投多少才不至于压力过大」
* ✅ 模拟 **600 / 800 / 1000 万资产阶段** 的提取能力
* ✅ 明确什么情况下应该降档、暂停提取
* ✅ 避免只看分配金、忽略本金被吃掉的风险

这是很多人“投资多年却依然焦虑”的根本原因。
